TEI Annotator

This demo application annotates bibliographic information with TEI XML tags such as tei:author, tei:title etc. Paste a bibliographic entry below or run evaluation against test examples.

No provider is configured. Set at least one API key in webservice/.env and restart the server (HF_TOKEN, GEMINI_API_KEY, or KISSKI_API_KEY).
